🚀 If you are tired of writing complex system prompts from scratch,then use this AI application that writes prompt for you. 

💡 How it works:
Users input a raw, messy idea (e.g., "how to climb mount everest"). The application feeds this input into a custom Meta-Prompt running on the backend. It dynamically restructures the idea into a production-ready prompt using clear [Role], [Context], [Task], and [Constraints] boundaries.

🛠️ The Tech Stack:
• Frontend: HTML5, CSS3, and JavaScript (vanilla DOM handling)
• Backend: Node.js & Express API routing
• Network: Official Groq SDK leveraging high-speed inference models
• Infrastructure: Version control via GitHub, securely deployed on Render utilizing Environment Variables to protect secret keys.

Building this project taught me how to securely manage API gateways and translate user intent into structured model behavior
